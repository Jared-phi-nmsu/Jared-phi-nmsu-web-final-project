# Jared-phi-nmsu-web-final-project
This was a website for the client green tech solutions. This website has 5 main sections a home page, about us, contacts us, our products, and a blog. 
I used HTML CSS, and JS to complate this project, I was planning on using react but with the lack of pertinent information on the business it was easier to get a simple framework started.
All text is either lorem or was made with a text generator. All elements in the webpage were designed to scale with the size of the browser , rather the devices watching from. 
I used several pictures all are in the open free use domain. 
Both the products and tips are created in a way that enables the user to easily expand, alter and adjust those lists. I created the countdown clock and the carbon ticker to add a sense of 
urgency to the climate disaster as well as to show that the owners are passionate about carbon reduction and climate change prevention.
